预备公式，奇数相加：
1 + 3 + 5 + 7 ... + (2n-1) = n^2
-- n为奇数的个数
例如：
1 + 3 + 5 = 3 ^ 2 = 9

圆球每层放置文字数为奇数个，例如：1, 3, 5, 7, 9, 7, 5, 3, 1
假设一个圆球划分为2n-1层，则可容纳文字数为：
1 + 3 + 5 + ... + (2(n-1)-1) +(2n-1) + (2(n-1)-1) + ... + 5 + 3 + 1
= n^2 + (n-1)^2
